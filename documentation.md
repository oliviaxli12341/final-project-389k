## 1. Database:
  Data is stored using MongoDB via the mongoose npm package.

##### Schemas:

  postSchema:
```javascript
  {
    text: String,
    id: Number,
    time: Number,
    user: String,
    tags: [String],
    comments: [commentSchema]
  }
```
commentSchema:
```javascript
{
  text: String,
  time: Number,
  user: String
}
```
UserSchema:
```javascript
{
  user: String,
  description: String,
  picture: String
}
```

## 4. API:

  * Post Endpoints:
    * /user/:name/edit
    * /post/:id/comments
  * Delete Endpoints:
    * /post/:id/delete  (deletes a specific post according to id - the post number)
    * /user/post/:name/delete  (deletes posts associated with a user, doesn't delete the user)
    * /user/:name/delete  (deletes the user)
